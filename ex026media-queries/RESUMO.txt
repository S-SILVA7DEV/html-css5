Basicamente, para mudar a orietação (tela em pé ou deitada) os computadores vão pegar pela resolução da tela.

se a altura tiver maior que a largura, ele está como 'portrait' (retrato)
se a largura tiver maior que a altura, ele esta como 'landscape' (paisagem)
_____________________________________________________________________________

para ficar mais facil de entender e ficar, imagine voce tirando um foto de uma PAISAGEM (landscape), voce tiraria a foto com o celular DEITADO, certo?
E se você estivesse tirando um retrato (tipo uma selfie). fica bem mais fácil com o celular em pé (portrait)

a configuração media="all" vai servir para todos os dipositivos, pois algumas configuraçõe se repetem.

<link rel="stylesheet" href="estilos/style.css"media=all">

media types são escritos dentro das aspas normais de media=" "
ex:	 media="screen" tela
	 media="print" impressora
	 media="all" configurações gerais

mas as medias features são escritas dentro das aspas e dentro de (parênteses), logo após os media types.
media="screen and (orientation: portrait)"

	<link rel="stylesheet" href="estilos/retrato.css"media="screen and (orientation: portrait)">

	<link rel="stylesheet" href="estilos/paisagem.css"media="screen and (orientation: landscape)">
____________________________________________________________________

importante: media types não tem parênteses, media features tem parênteses isoladamente, ou seja, se tivermos três media fetures, teremos 3 entre parênteses
